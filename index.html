New here? Start with one exercise.

You do not need to know the tools, or any of the frameworks. Policy reasoning is a craft. You learn it by doing it once, then doing it again.

  1. Pick a policy issue from this week's news — something you already have an opinion about.
  2. Run it through Policy Analysis. It walks you through eight steps and ends with a memo. The tool guides you; you do not need to know the method going in.
  3. Run Find a Framework on the same issue. Seeing which lenses apply to a problem you understand is the fastest way to build analytical vocabulary.

Want the thinking behind all this first? Read how Scholar thinks, or browse the full framework library at frameworks.pranaykotas.com.
